== Description ==

The Bitcoin Plus Miner lets you make money by having your visitors generate bitcoin. This can be done in the background with the hidden mode, or show up as a widget so your visitors can start/stop the bitcoin miner.

== Installation ==

1. Upload the entire `bitcoin-plus-miner` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar, filling in your email address.
